## UI元素优化

### 1. 植物卡片美化

#### 1.1 卡片设计规范

| 元素 | 规格 | 说明 |
|------|------|------|
| 卡片尺寸 | 60x80 px | 标准尺寸 |
| 圆角半径 | 5 px | 圆角矩形 |
| 边框宽度 | 2 px | 边框 |
| 内边距 | 4 px | 内容内边距 |

#### 1.2 植物类型配色方案

| 植物类型 | 主色 | 辅色 | 示例植物 |
|---------|------|------|---------|
| 射手类 | (34, 139, 34) | (50, 205, 50) | 豌豆射手、双发射手 |
| 防御类 | (139, 90, 43) | (160, 82, 45) | 坚果墙、高坚果 |
| 爆炸类 | (220, 20, 60) | (255, 100, 50) | 樱桃炸弹、土豆雷 |
| 寒冰类 | (135, 206, 235) | (200, 230, 255) | 寒冰射手、冰西瓜 |
| 投手类 | (139, 69, 19) | (205, 133, 63) | 西瓜投手、玉米投手 |
| 蘑菇类 | (128, 0, 128) | (180, 100, 180) | 小喷菇、大喷菇 |
| 特殊类 | (255, 140, 0) | (255, 200, 50) | 磁力菇、仙人掌 |

#### 1.3 卡片状态视觉

| 状态 | 视觉效果 | Alpha值 |
|------|---------|---------|
| 可种植 | 亮色，完整显示 | 255 |
| 冷却中 | 半透明 + 冷却进度条 | 150 |
| 阳光不足 | 灰色显示 | 100 |
| 选中 | 高亮边框 + 放大效果 | 255 |

### 2. 阳光计数器优化

#### 2.1 设计规范

- 位置：右上角
- 尺寸：80x40 px
- 背景：半透明棕色圆角矩形
- 字体：加粗，金色
- 数字动画：收集阳光时跳动效果

## 植物卡片美化

### 3.1 卡片层级结构

```
┌─────────────────┐
│   [植物图标]    │ ← 植物精灵（40x40）
│                 │
│   植物名称      │ ← 名称文字（居中）
│   [阳光消耗]    │ ← 阳光图标 + 数字
└─────────────────┘
```

### 3.2 冷却动画

- 垂直进度条
- 从下往上填充
- 半透明遮罩
- 进度文字显示

### 3.3 选中效果

- 边框发光
- 轻微放大（1.1倍）
- 上移效果（+5 px）

## 视觉规范

### 5.1 颜色规范

| 用途 | 颜色值 |
|------|---------|
| 主背景 | (20, 30, 50) |
| UI背景 | (50, 50, 70, 200) |
| 边框 | (100, 100, 120) |
| 文字（主要） | (255, 255, 255) |
| 文字（次要） | (200, 200, 200) |
| 强调色（金色） | (255, 215, 0) |
| 强调色（绿色） | (50, 205, 50) |
| 强调色（红色） | (220, 20, 60) |

### 5.2 字体规范

| 用途 | 字号 | 样式 |
|------|------|------|
| 标题 | 32 px | 加粗 |
| 副标题 | 24 px | 加粗 |
| 正文 | 16 px | 正常 |
| 小字 | 12 px | 正常 |

### 5.3 间距规范

| 元素 | 间距 |
|------|------|
| 按钮之间 | 20 px |
| 卡片之间 | 10 px |
| 文字与边框 | 8 px |
| 界面边距 | 30 px |

### 5.4 动画规范

| 动画 | 时长 | 缓动函数 |
|------|------|---------|
| 按钮悬停 | 0.15 s | ease-out |
| 页面切换 | 0.3 s | ease-in-out |
| 弹窗出现 | 0.2 s | ease-out |
| 进度更新 | 0.1 s | linear |
