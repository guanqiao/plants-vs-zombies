## 使用方式

代码已自动配置为优先加载这些图片资源。如果图片不存在，会回退到使用程序生成的占位符。

## 扩展资源

如需添加更多资源：

1. **直接添加图片**: 将图片放入对应目录，然后在 `src/arcade_game/entity_factory.py` 中更新映射配置

2. **使用生成脚本扩展**: 修改 `tools/generate_assets.py` 添加新的生成函数

3. **使用外部美术资源**: 可以替换为更精美的像素画或手绘资源，只要保持相同的文件名和尺寸

## 注意事项

- 所有图片使用 PNG 格式，支持透明背景
- 建议保持统一的像素风格
- 动画资源可以通过添加多帧图片实现
