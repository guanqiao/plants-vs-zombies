
# 音效文件说明

本目录存放游戏所需的音效文件。

## 音效文件列表

请将以下音效文件放置在此目录中：

| 文件名 | 描述 |
|--------|------|
| `plant.wav` | 种植植物音效 |
| `shoot.wav` | 射击音效 |
| `hit.wav` | 击中音效 |
| `collect_sun.wav` | 收集阳光音效 |
| `zombie_death.wav` | 僵尸死亡音效 |
| `zombie_eat.wav` | 僵尸吃植物音效 |
| `game_over.wav` | 游戏结束音效 |
| `victory.wav` | 胜利音效 |
| `button_click.wav` | 按钮点击音效 |
| `explosion.wav` | 爆炸音效 |
| `cherry_bomb.wav` | 樱桃炸弹音效 |
| `potato_mine.wav` | 土豆雷音效 |
| `ice_hit.wav` | 冰冻击中音效 |
| `fire_hit.wav` | 火焰击中音效 |
| `splash.wav` | 溅射音效 |

## 背景音乐

请将背景音乐文件放置在 `assets/music/` 目录中：

| 文件名 | 描述 |
|--------|------|
| `main_menu.wav` | 主菜单音乐 |
| `game_play.wav` | 游戏进行中音乐 |

## 音效格式

支持的音频格式：
- WAV
- OGG
- MP3（某些平台可能需要额外的依赖）

## 占位模式

如果音效文件不存在，游戏会静默运行（无音效），不会报错。
